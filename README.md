# Tadabbur

**An AI-powered Qur'an companion for reflection, guidance, and daily inspiration.**

## Overview

Tadabbur is a mobile application designed to help users explore and reflect on the teachings of the Qur'an through an intuitive and modern experience. The application combines AI-powered conversations with essential Islamic features to make learning, reflection, and daily engagement with the Qur'an more accessible.

Users can ask questions related to the Qur'an, receive relevant verses with translations and explanations, save meaningful verses for future reference, and view prayer times based on their current location.

---

## Features

### AI Qur'an Chat
- Ask questions related to the Qur'an and Islamic guidance.
- Receive relevant Qur'anic verses in Arabic.
- View English translations and concise explanations.
- Designed to encourage reflection through Qur'anic teachings.

### Saved Verses
- Save meaningful verses for future reference.
- Maintain a personal collection of bookmarked verses.
- Easily revisit previously saved verses.

### Prayer Times
- View daily prayer times based on the user's current location.
- Stay informed about upcoming prayers throughout the day.

### User Authentication
- Secure sign up and login.
- Personalized experience with access to saved verses across sessions.

## Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/tadabbur.git
cd tadabbur
```

### Install Frontend

```bash
cd frontend
npm install
npx expo start
```

### Install Backend

```bash
cd backend
npm install
node server.js
```
